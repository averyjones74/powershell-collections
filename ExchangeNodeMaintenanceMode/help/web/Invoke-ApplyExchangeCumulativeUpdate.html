Invoke-ApplyExchangeCumulativeUpdate

Synopsis

Apply an Exchange Cumulative Update

Syntax

Invoke-ApplyExchangeCumulativeUpdate [-Source] [-Prepare] [-UMLangHandling] [-UMLangSource] [-UMLanguages] [<CommonParameters>]

Detailed Description

Apply an Exchange Cumulative Update, with the optional AD and Schema update, and an optional UM language Update.

Parameters
